## 如何定义变量？

为啥要定义变量？

- 他是用来把一些数据暂时存起来，不然等下找不到就不好处理。所以就有变量，可以吧变量想象成一个盒子，数据就是一些实际的东西，然后定义一个变量就是去拿一个盒子，但是这个盒子的大小是不限制的。有了盒子之后就可以往里面装东西了：

```js
// 定义一个变量(盒子) box 来放一些东西（数据）,但是一开始我们不放任何数据
let box = ''		// 此时 box的值为 ''
// 定义的时候前面加了 ‘let’， 如果是let定义的字符串，那么我们可以随心所欲的给他替换成任何的值,比如我们做这个操作
box = '你是一个盒子'	// 此时 box的值为 '你是一个盒子'
// 因为上面使用 let 定义了这个盒子；然后我们就可以直接拿到这个盒子给他换成别的东西
// 当然你还可以把它定义成别的东西
box = ['a', 'b', 'c'] 	// 此时box的值为 ['a', 'b', 'c']
// 也就是说，使用let定义的盒子(变量)，是可以随便怎么放东西，随心所欲的！
```

在这个时候，把数据类型搬上来大好不过了，不用记下来，看一眼知道有那么回事即可！需要的时候对着表看就行

- 数字（number）：就是不带“”引号的数字 2131
- 字符串（string）：就是带引号的任何内容 “asdafakfkas法芙娜时间9329愉快”
- 布尔值（boolean）： 只有 true | false 两种，当然 0表示 false ''也表示false，undefined 、 null都表示为false
- undefined：就是没找着这个值或者这个值不存在呗
- null：值为空呗
- 对象（Object）：刚刚讲的很细，就是用来描述一个东西的详细信息
- 数组： ['x','x', ....] 这就是数组

当然还有更多，最常用的就这些。
