        # Vue.js 是什么

        Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

        # 运行GitHub上的Vue项目

        通过GitHub下载相应的Vue项目到本地，解压后进入到文件夹，通过cmd进入到相关目录后运行如下命令：

        npm install
        1

        这样就会安装好项目的相关依赖。然后输入运行项目命令：

        npm run dev
        1

        # 模块的安装

        安装模块可以很便捷的对项目进行功能的完善，快速的开发我们的项目。

        我们以安装模块axios为例，首先我们打卡目录下的命令模式

        npm i axios --save
        1

        在命令输入完成后下载结束后即可完成模块的相关安装，此时我们会在package.json文件中的dependencies中会多出我们刚刚安装好的模块名称，以及版本号。

        如果需要指定版本号进行安装，则

        npm i axios@版本号 --save
        1

        # 卸载模块

        还是以axios为例子

        npm uninstall axios
        1

        # 安装CNPM提高下载速度

        使用定制的 cnpm (gzip 压缩支持) 命令行工具代替默认的 npm

        npm install -g cnpm --registry=https://registry.npm.taobao.org
        1

        接下来，就可以使用cnpm来代替之前的npm进行命令的输入了。
