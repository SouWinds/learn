    # 一般方法创建Node.js项目

    # 核心命令

    npm init
    1

    使用NPM安装依赖：express模块

    npm install express --save
    1

    创建好index.js文件之后，输入以下内容

    var express = require('express');
var app = express();

app.get('/', function(res, rep) {
    rep.send('Hello, word!');
});

app.listen(3000);

    全局安装nodemon使node项目在修改后自动重启

    npm i nodemon -g
    1

    运行命令开启项目
